### 3.SUMIFS 按条件求和  
=SUMIFS(L2:L8,D2:D8,D2)  
L2:L8是要求和的数字  D2:D8是判断域  D2是判决标准  （也能写成{ , , }，但是不能选择一列，所以可以先选出F,Y辅助列，然后再求和也行）
